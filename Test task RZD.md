**Grafana**
```bash
sudo docker run -d -p 3000:3000 \
  -v grafana-storage:/var/lib/grafana \
  --name=grafana \
  grafana/grafana
```

**cAdvisor**
```bash
sudo docker run -d \
  --name=cadvisor \
  --volume=/:/rootfs:ro \
  --volume=/var/run:/var/run:rw \
  --volume=/sys:/sys:ro \
  --volume=/var/lib/docker/:/var/lib/docker:ro \
  -p 8080:8080 \
  gcr.io/cadvisor/cadvisor:latest
```
